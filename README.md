# 🎯 KBC Quiz Game (Java)

A simple console-based **Kaun Banega Crorepati (KBC) Quiz Game** developed using Java. The game asks multiple-choice questions, provides lifelines, and rewards players for correct answers.

## 🚀 Features

- 10 Multiple Choice Questions
- Reward System (Prize Money)
- 3 Lifelines:
  - 50-50
  - Audience Poll
  - Skip Question
- Quit Option
- Jackpot Winner Announcement
- Safe Winnings Logic

## 🛠 Technologies Used

- Java
- Scanner Class
- Arrays
- Loops
- Conditional Statements

## 📋 Game Rules

1. Answer questions by choosing options 1-4.
2. Enter **5** to use a lifeline.
3. Enter **6** to quit the game.
4. Lifelines can be used only once.
5. Lifelines are not available for Question 9 and Question 10.
6. Wrong answers result in game over with safe winnings.

## 🎮 Available Lifelines

### 50-50
Removes incorrect options and helps narrow down the answer.

### Audience Poll
Shows the audience's most voted answer.

### Skip Question
Skips the current question and moves to the next one.
